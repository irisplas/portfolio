## Projecten

### 1. Smaakvol — Recepten beheer app

**Type:** Fullstack webapplicatie · Schoolproject (examenopdracht, team van 2)  
**Status:** Afgerond · Live

Een volledige recepten-webapplicatie waarbij gebruikers recepten kunnen aanmaken, bekijken, bewerken en als favoriet markeren. Recepten kunnen uitgerust worden met instructies, ingrediënten met hoeveelheden, en een foto.

**Technologieën**

| Laag | Stack |
|------|-------|
| Frontend | React, Vite, Tailwind CSS |
| Backend | NestJS, Drizzle ORM, MySQL |
| Testing | Cypress (e2e), Jest |
| Extra | Multer (file upload), Swagger (API docs), Docker |
| Deploy | Render |

**Highlights**
- REST API met volledige CRUD voor recepten, ingrediënten en gebruikers
- JWT authenticatie met rol-gebaseerde toegangscontrole (admin/user)
- Foto-upload via Multer
- End-to-end testen met Cypress
- Automatische API-documentatie via Swagger
- Docker-ondersteuning voor lokale ontwikkeling

**Links**
-  [Live frontend](https://smaakvol.onrender.com)
-  [Live backend API](https://frontendweb-2526-plaswicke.onrender.com)
-  [GitHub repository](https://github.com/HOGENT-frontendweb/frontendweb-2526-PlasWicke)

### 3.  FIFA World Cup App — Toernooi tracker

**Type:** Java webapplicatie · Schoolproject  
**Status:**  Afgerond

Een webapplicatie voor het bijhouden en weergeven van FIFA World Cup data en het ingeven van prognoses. Gebouwd met Java Spring Boot en Thymeleaf server-side templating.

**Technologieën**

| Laag | Stack |

| Backend | Java, Spring Boot |

| Database | MySQL |

**Highlights**
- Server-side rendering via Thymeleaf templates
